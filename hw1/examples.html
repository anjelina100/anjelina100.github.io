Examples & Tools

Common Visualization Examples

These are popular tools used to create charts and dashboards. Each example includes a type of visualization, the tool that can make it, and how you can access that tool.

Bar Chart - Excel / Google Sheets

Access: Excel is a desktop app; Google Sheets is a website.

Dashboard - Tableau

Access: Tableau Desktop is installed; Tableau Public is web-based.

Scatter Plot - Python (Matplotlib)

Access: Free library you run in Python (locally or in Google Colab).

Interactive Web Chart - D3.js

Access: JavaScript library used in web pages (runs in the browser).
